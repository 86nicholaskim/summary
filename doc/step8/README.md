# 🧩 Step 8: 데스크탑 앱 전환 (Desktop Body)

사만다와 에디터를 **하나의 기기 전용 어플리케이션**으로 완성하는 단계입니다.

---

## 🎯 목표
1. **Electron 패키징**: 웹 기술을 데스크탑 환경으로 통합.
2. **하드웨어 제어**: AI 에이전트가 카메라를 켜고 파일을 읽는 물리적 능력 확보.
3. **전문 인쇄 시스템**: A4 규격에 완벽히 대응하는 인쇄 및 PDF 저장.

## 📝 세부 작업 리스트
- [ ] **8.1 Electron Setup**: 모노레포 내 Electron 메인 프로세스 구축.
- [ ] **8.2 Camera Integration**: AI가 직접 카메라 스냅샷을 찍는 기능.
- [ ] **8.3 Local File System**: AI의 로컬 파일 직접 읽기/쓰기 권한 부여.
- [ ] **8.4 Professional Printing**: 페이지 레이아웃 및 여백 자동 조정 시스템.

## 💡 핵심 기술
- Electron.js (Main/Renderer)
- WebRTC (Camera)
- Native File I/O
