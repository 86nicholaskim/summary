# 🧩 Step 6: 시각 지능 통합 (Visual Intelligence)

사만다에게 **이미지와 서식을 읽을 수 있는 눈**을 달아주는 단계입니다.

---

## 🎯 목표
1. **Native Vision 활용**: Gemma 4의 시각 지능을 이용한 레이아웃 분석.
2. **데이터 구조 추출**: 사진 속 표(Table)와 텍스트 항목을 완벽히 식별.
3. **엑셀 서식 파싱**: 복잡한 엑셀 시각 데이터를 구조화된 JSON으로 변환.

## 📝 세부 작업 리스트
- [ ] **6.1 Image Pipeline**: 이미지 업로드 및 Base64 인코딩 전송 모듈.
- [ ] **6.2 Layout OCR**: 종이 서식의 구역(Header, Body, Footer) 구분.
- [ ] **6.3 Table Recognizer**: 표의 행(Row)과 열(Column) 개수 및 내용 추출.
- [ ] **6.4 Excel Visualizer**: 엑셀 캡처 이미지를 보고 서식 데이터 복원.

## 💡 핵심 기술
- Gemma 4 Multimodal (Vision)
- Base64 Encoding
- Structural Data Extraction
